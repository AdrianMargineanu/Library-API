README
--------------
Fisier request.c contine functiile ce creaza un anumit tip de request are este transmit catre server.
Conexinea este reinnoita inainte de trimiterea unei noi comezi.
Pentru a interpreta mesajele json am utilizat libraria "Parson.h", Deoarece fucntiile ce se pot gasi 
in aceasta librarie sunt foarte usor de utilizat, pentru a extrage sau insera informatia intr-un mesaj de tip json.

Tema poate fi rulata cu "make run" ce va arata functionalitatea normala a programului, dar si cu "make run-debug" 
pentru a vedea mesajele transmise si primite de la server.
